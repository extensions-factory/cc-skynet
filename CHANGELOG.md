# Changelog

All notable changes to this project will be documented in this file.

Format follows [Keep a Changelog](https://keepachangelog.com/).

## [0.4.0] - 2026-03-29

### Added

- `init-project` skill — initialize a project workspace with `.skynet/project.json`
- `detect-project` skill — detect and display an initialized Skynet project
- Project config schema documentation (`docs/architecture/project-config-schema.md`)

## [0.3.0] - 2026-03-29

### Added

- One-command install script (`scripts/install.sh`)
- `check-version` skill for post-install verification
- Install troubleshooting guide (`docs/guides/install-guide.md`)

## [0.2.0] - 2026-03-28

### Added

- Skynet orchestrator agent definition (`plugins/skynet/agents/skynet.md`)
- Plugin settings (`plugins/skynet/settings.json`)
- README with installation and usage instructions

## [0.1.0] - 2026-03-21

### Added

- Initial marketplace scaffold
- Project structure and conventions
